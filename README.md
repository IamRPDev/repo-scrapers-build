# Scrapers Builder 🏗️

This repository is the automated **Builder** for **Scrapers**.

## Ecosystem Role
Part of the [Kodi Build Hub](https://github.com/IamRPDev/kodi-build). This repo hosts its own releases locally.

## Build Structure
The build process strictly follows this standardized structure:
- **Source**: Upstream code is checked out into `./source/repo-scrapers/`.
- **Output**: Compiled binaries are organized into `./compiled/<os>/<version>/`.
- **Artifacts**: Final files use the naming convention `repo-scrapers-<os>-<version>-<branch>.[zip|tar.gz]`.
